## Como Rodar o programa:

* Execute o dockerdesktop no seu pc (baixe, se for preciso)
* `docker-compose up -d`
* `npm install`
* `npm install -D tsx` 
* `npx prisma migrate deploy`
* `npx prisma generate`
* `npx prisma db seed`
* `npm run dev`
* Pra usar o prisma studio, split o terminal e rode `npx prisma studio`
